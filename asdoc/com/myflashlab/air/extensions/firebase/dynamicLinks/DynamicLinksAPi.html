public function build($appCode:String, $link:String, $makeShort:Boolean, $androidParams:AndroidParams, $iosParams:IosParams, $socialMediaParams:SocialMediaParams, $analyticsParams:AnalyticsParams, $onComplete:Function):void

This method will let you create a dynamic link and the generated deeplink will be passed through the function you would specify.

Before being able to use this method, you must have set Firebase.getConfig().webApiKey property manually.

Parameters

$appCode:String — Required
 
$link:String — Required
 
$makeShort:Boolean
 
$androidParams:AndroidParams — Optional - pass null if you don't want to set it.
 
$iosParams:IosParams — Optional - pass null if you don't want to set it.
 
$socialMediaParams:SocialMediaParams — Optional - pass null if you don't want to set it.
 
$analyticsParams:AnalyticsParams — Optional - pass null if you don't want to set it.
 
$onComplete:Function — Required - When the deeplink is generated, it will call this Function.


Example
The following sample shows how you can generate deeplinks with Firebase DynamicLinks standards. 
         var androidParams:AndroidParams = new AndroidParams();
         androidParams.apn = "air." + NativeApplication.nativeApplication.applicationID;
         androidParams.amv = 3;
         
         var iosParams:IosParams = new IosParams();
         iosParams.ibi = NativeApplication.nativeApplication.applicationID;
         iosParams.isi = "1094775775";
         
         var socialMediaParams:SocialMediaParams = new SocialMediaParams();
         socialMediaParams.st = "my Social Media title";
         socialMediaParams.sd = "my Social Media description";
         
         DynamicLinks.api.build("r28kt", "http://www.myflashlabs.com/deeplinks", true, androidParams, iosParams, socialMediaParams, null, onDeeplinkBuiltDone);
         
         function onDeeplinkBuiltDone($deeplink:String, $raw:String):void
         {
             if ($raw)
             {
                 trace("$raw = " + $raw);
             }
             
             if($deeplink)
             {
                 trace("_deepLink = " + $deeplink);
             }
         }
