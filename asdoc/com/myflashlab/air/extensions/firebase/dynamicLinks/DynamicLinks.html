api:DynamicLinksAPi  [read-only]

Use this method to have access to the Firebase DynamicLinks REST API for generating deeplinks on the fly.

NOTICE: THIS FEATURE IS ONLY AVAILABLE TO MYFLASHLABS CLUB MEMBERS. If you are a club member, you can use this method to have easy access to DynamicLinks REST API to build deeplinks with Firebase standards. If you are not a club member though, you can still go to your Firebase console and build dynamic links there. Or use REST API yourself and generate deeplinks. However by Joining the club, you are supporting the MyFlashLabs Team to be able to do their job with more ease.
