Packagecom.myflashlab.air.extensions.streamPlayer
Classpublic class StreamPlayerFrame
InheritanceStreamPlayerFrame flash.events.EventDispatcher

This class must be initialized using StreamPlayer.getInstance();. When initialized, you need to listen to StreamPlayerEvents.FRAME_INITIALIZED events to make sure the native window has been created successfully. After that, you can use the instance.
