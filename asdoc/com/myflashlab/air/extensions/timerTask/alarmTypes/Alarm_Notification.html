Classpublic class Alarm_Notification
InheritanceAlarm_Notification Alarm_Types flash.events.EventDispatcher

This class will let you create a new notification task. as you may know already, notifications on Android has many different settings and a lot of them are not supported on older Android versions but with the help of this class, you don't have to worry about different Android versions. any feature which may not be supported on older devices will be safely ignored and it won't break your app.

setTouchAction()method 
public function setTouchAction($obj1:Object, $obj2:Object = null, $obj3:Object = null, $obj4:Object = null):void

This method will help you specify what action must be executed when user touches the notification. the current version of this extension supports two actions: 1) open the app. 2) open device browser

if you are going to open the app, you will have the option to pass a string so when the user opens your app, this string can be fetched and you can do different operations in your app based on the coming information. on the other hand, you can open a URL in device's browser.

the first object for this method must always be the action for the main notification itself but the other three objects are optional. you may use the TouchActionObject class to build your objects

Parameters

$obj1:Object
 
$obj2:Object (default = null)
 
$obj3:Object (default = null)
 
$obj4:Object (default = null)

See also

com.myflashlab.air.extensions.timerTask.alarmTypes.alarm_Notification.TouchActionObject

Example
below is a sample showing you how to create objects for touch actions: 
         var actionObj1:TouchActionObject = new TouchActionObject();
         actionObj1.setSelfType();
         actionObj1.setAppAction("Message to be sent to flash app when user touches the notification!");
         
         var actionObj2:TouchActionObject = new TouchActionObject();
         actionObj2.setBtnType(0x0108001f, "myappsnippet");
         actionObj2.setBrowseAction("http://www.myappsnippet.com/");
         
         var actionObj3:TouchActionObject = new TouchActionObject();
         actionObj3.setBrowseAction("http://www.myflashlab.com/");
         actionObj3.setBtnType(0x0108001a, "myflashlab");
         
         var actionObj4:TouchActionObject = new TouchActionObject();
         actionObj4.setBrowseAction("http://www.google.com/");
         actionObj4.setBtnType(0x0108001a, "google");
         
         // and finally send them into your task
         alarmTask.setTouchAction(actionObj1, actionObj2, actionObj3, actionObj4);
