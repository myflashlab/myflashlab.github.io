addEventListener()method
override public function addEventListener(type:String, listener:Function, useCapture:Boolean = false, priority:int = 0, useWeakReference:Boolean = false):void

Use this method to listen to general Admob events like:

  • AdMobEvents.AD_CLOSED
  • AdMobEvents.AD_FAILED
  • AdMobEvents.AD_LEFT_APP
  • AdMobEvents.AD_LOADED
  • AdMobEvents.AD_OPENED

Every Ad type, may have some other events which you may find helpful to listen to. For example the Banner Ads class has a BannerEvents.SIZE_MEASURED event that will let you know the banner size in pixels.
