Packagecom.myflashlab.air.extensions.richWebView
Classpublic class RichWebView
InheritanceRichWebView Object

This extension is a perfect replacement to the classic StageWebView and it allows you to easily call Javascript functions from Air and send String messages from JS to Air. it also gives you many new features that the classic StageWebView couldn't provide. Features like File pick or GPS access and many more.

ENABLE_AIR_PREFIXproperty 
public static var ENABLE_AIR_PREFIX:Boolean = true

(Android ONLY) Set this property to false if you have forced your AIR app to drop the "air." prefix when packaging the .apk file

Default value is true. Notice that all AIR applications will have the "air." prefix added to their package name on Android automatically. If you are manually removing the "air." prefix, you need to set this property to false. otherwise leave it as it is.
