BG_COLOR_HEXproperty
public static var BG_COLOR_HEX:String = #FFFFFFFF

Set a background color for your webview content in hex #AARRGGBB format. The first two digits will represent the alpha channel of your webview background color.

Default value is set to #FFFFFFFF

ENABLE_AIR_PREFIXproperty 
public static var ENABLE_AIR_PREFIX:Boolean = true

(Android ONLY) Set this property to false if you have forced your AIR app to drop the "air." prefix from your Android package name

Default value is set to true. Notice that all AIR applications will have the "air." prefix added to their package name on Android automatically. If you are manually removing the "air." prefix, you need to set this property to false. otherwise leave it as it is.

ENABLE_BITMAP_CAPTUREproperty 
public static var ENABLE_BITMAP_CAPTURE:Boolean = false

You must set this property to true if you want to take screenshot from your web content. If you don't need to take screenshots, set it to false for better performance.

Default value is set to false
