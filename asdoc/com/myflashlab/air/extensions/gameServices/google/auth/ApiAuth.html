login()method 
public function login():Boolean

Call this method so the ANE will start trying to login the user to the Game Services.

NOTE: When you login a user to the Game Services, you don't have to ever log them out (unless you have a good reason to do that). Not logging users out explicitly, will let the ANE to login silently when you first run your game. This is a very important user-friendly fact and will avoid troubling users to manually hit the login button each time they enter your app.

Returns
Boolean — returns true if the ANE has successfully started to login you to the service. However, it does not mean that you have logged in! it just means that the ANE is able to go ahead with this operation. In case the returned value is false, you must have listened to the AuthEvents.ERROR event to know what the problem could have been.

logout()method 
public function logout():Boolean

Call this method to logout from the Game Services. When you logout using this method, the AuthEvents.TRYING_SILENT_LOGIN event will always tell you that the user cannot login silently and you have to call the login gain.

Returns
Boolean — returns true if the ANE was able to successfully log you out. In case the returned value is false, you must have listened to the AuthEvents.ERROR event to know what the problem could have been.
