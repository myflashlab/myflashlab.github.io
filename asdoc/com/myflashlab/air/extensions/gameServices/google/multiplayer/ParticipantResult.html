ParticipantResult()Constructor
public function ParticipantResult($participantId:String, $result:int, $placing:int)

Create a new participant result object by initializing it using this constructor method.

Parameters
$participantId:String — The ID of the participant this result is for.
 
$result:int — The result type for this participant in the match. One of MATCH_RESULT_WIN, MATCH_RESULT_LOSS, MATCH_RESULT_TIE, MATCH_RESULT_NONE, MATCH_RESULT_DISCONNECT, or MATCH_RESULT_DISAGREED.
 
$placing:int — The placing of this participant in the match. Use PLACING_UNINITIALIZED to indicate that no placing should be reported.
