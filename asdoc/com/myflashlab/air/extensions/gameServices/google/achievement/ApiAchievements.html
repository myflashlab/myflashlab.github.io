increment()method
public function increment($id:String, $numSteps:int, $immediate:Boolean = false):void

Increments an achievement by the given number of steps. The achievement must be an incremental achievement. Once an achievement reaches at least the maximum number of steps, it will be unlocked automatically. Any further increments will be ignored.

If the $immediate parameter is set to false, This is the fire-and-forget form of the API. Use this form if you don't need to know the status of the operation immediately. For most applications, this will be the preferred API to use, though note that the update may not be sent to the server until the next sync.

if you need the operation to attempt to communicate to the server immediately or need to have the status code delivered to your application, set $immediate to true.

Parameters

$id:String — The achievement ID to increment.
 
$numSteps:int — The number of steps to increment by. Must be greater than 0.
 
$immediate:Boolean (default = false)

setSteps()method 
public function setSteps($id:String, $numSteps:int, $immediate:Boolean = false):void

Set an achievement to have at least the given number of steps completed. Calling this method while the achievement already has more steps than the provided value is a no-op. Once the achievement reaches the maximum number of steps, the achievement will automatically be unlocked, and any further mutation operations will be ignored.

If the $immediate parameter is set to false, This is the fire-and-forget form of the API. Use this form if you don't need to know the status of the operation immediately. For most applications, this will be the preferred API to use, though note that the update may not be sent to the server until the next sync.
