Property Detail
acceptedTimestampproperty
acceptedTimestamp:Date  [read-only]

Retrieves the timestamp (in milliseconds since epoch) at which this quest was accepted by the player. If the quest has never been accepted, this will return Quest.UNSET_QUEST_TIMESTAMP. If you are looking for the time that a quest is available to be accepted, see getStartTimestamp(). This value should always be greater than getStartTimestamp and less than getEndTimestamp
