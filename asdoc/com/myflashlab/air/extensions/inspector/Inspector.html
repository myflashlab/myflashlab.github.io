check()method
public static function check($class:Class, $platformCheck:Boolean = true, $dependencyCheck:Boolean = false):Boolean

Call this method while passing the ANE class which you wish to inspect. This method can check the target ANE for two aspects. 1) checks if the current running platform meets the minimum requirements needed by the target ANE. 2) checks if the target ANE has access to its dependencies (if any)

NOTE 1: The default value for the $dependencyCheck parameter is false. If you need to check the dependencies, you need to set it to true. However, you need to remember that the operation to check dependencies is very RAM hungry! So when you are releasing your app, make sure to turn it off.

NOTE 2: Unlike the $dependencyCheck parameter, the $platformCheck is very light and you can safely keep it when releasing your app.

NOTE 3: $dependencyCheck is something that is in your control when developing the ANE so there is absolutly no need to keep it active when you are releasing your app. On the other hand, $platformCheck checks the platform which the app is running on. You don't have any control over what platform your app might run on by different users. So, you do need to keep the $platformCheck parameter active always.
